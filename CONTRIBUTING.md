# Contributing to Superai

Thanks for your interest in contributing.

## Development Setup

1. Install dependencies:

```bash
npm install
```

2. Start Metro:

```bash
npm start
```

3. Run iOS or Android:

```bash
npm run ios
npm run android
```

## Contribution Guidelines

- Use TypeScript for all new files.
- Keep architecture modular (`screens`, `components`, `services`, `hooks`, `utils`).
- Add meaningful error handling for network and storage operations.
- Keep UI clean and minimal.
- Avoid collecting user data or adding telemetry.

## Pull Request Checklist

- App compiles and runs on iOS and Android
- No TypeScript errors
- No lint errors
- README updated if behavior changed
- Privacy requirements are preserved

## Code of Conduct

Be respectful and constructive. Focus feedback on code and user impact.
