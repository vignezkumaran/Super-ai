# Privacy Policy

Rocket is designed as a privacy-first app.

## What We Collect

- We do not collect analytics.
- We do not track usage.
- We do not upload conversations to any Rocket-managed server.

## What Is Stored On Device

- API keys entered by the user
- App settings
- Conversation history

This data is stored locally on your device using AsyncStorage.

## Third-Party Providers

If you use cloud mode, your prompt is sent directly to the provider you selected:

- OpenAI
- Anthropic Claude

Rocket does not proxy cloud requests through its own backend.

## Local Mode

Local mode uses Ollama running on your own machine/network endpoint. Prompts are sent to your configured Ollama host.

If you choose to download open-source model files from providers like Hugging Face, downloads happen directly from that provider to your device.

## App Store Privacy Summary

- Data used to identify you: None
- Data linked to you: None
- Tracking: None
- Diagnostics/analytics SDKs: None

## Contact

Open an issue in the repository for privacy concerns or recommendations.
